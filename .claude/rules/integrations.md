# Правила разработки интеграций

> Применяется к: HTTP-сервисы, REST-клиенты, веб-сервисы, файловый обмен

## Принципы

1. **Сначала искать** — проверить существующие решения в БСП через `ssl_search`
2. **Согласовать контракт** — метод, URL, формат, аутентификация, таймауты, политика ретраев
3. **Безопасность** — никаких учётных данных в коде
4. **Идемпотентность** — повторный вызов не должен дублировать операцию

---

## Перед написанием кода

1. **`ssl_search`** — проверить БСП на наличие готовых подсистем интеграции
2. **`codesearch`** — найти существующие интеграции в конфигурации
3. **Согласовать контракт с пользователем:**
   - Метод и URL/endpoint
   - Формат запроса/ответа
   - Схема аутентификации
   - Таймауты и политика повторов
   - Требования к логированию

## HTTP-клиент

### Базовый шаблон

```bsl
Функция ВыполнитьHTTPЗапрос(Метод, Путь, ТелоЗапроса = Неопределено)

    Результат = Новый Структура("Успех, КодСостояния, Данные, Ошибка",
        Ложь, 0, Неопределено, "");

    // 1. Подключение с явными таймаутами
    Соединение = Новый HTTPСоединение(
        Сервер, Порт,
        , , , // Логин, пароль — из безопасного хранилища
        Таймаут);      // Обязательно!

    // 2. Формирование запроса
    HTTPЗапрос = Новый HTTPЗапрос(Путь);
    HTTPЗапрос.Заголовки.Вставить("Content-Type", "application/json");
    HTTPЗапрос.Заголовки.Вставить("Authorization", "Bearer " + Токен);

    Если ТелоЗапроса <> Неопределено Тогда
        HTTPЗапрос.УстановитьТелоИзСтроки(ТелоЗапроса, КодировкаТекста.UTF8);
    КонецЕсли;

    // 3. Выполнение с обработкой ошибок
    Попытка

        Если Метод = "GET" Тогда
            HTTPОтвет = Соединение.Получить(HTTPЗапрос);
        ИначеЕсли Метод = "POST" Тогда
            HTTPОтвет = Соединение.ОтправитьДляОбработки(HTTPЗапрос);
        ИначеЕсли Метод = "PUT" Тогда
            HTTPОтвет = Соединение.Записать(HTTPЗапрос);
        ИначеЕсли Метод = "DELETE" Тогда
            HTTPОтвет = Соединение.Удалить(HTTPЗапрос);
        КонецЕсли;

        Результат.КодСостояния = HTTPОтвет.КодСостояния;

        Если HTTPОтвет.КодСостояния >= 200 И HTTPОтвет.КодСостояния < 300 Тогда
            Результат.Успех = Истина;
            Результат.Данные = HTTPОтвет.ПолучитьТелоКакСтроку(КодировкаТекста.UTF8);
        Иначе
            Результат.Ошибка = СтрШаблон("HTTP %1: %2",
                HTTPОтвет.КодСостояния,
                HTTPОтвет.ПолучитьТелоКакСтроку(КодировкаТекста.UTF8));
        КонецЕсли;

    Исключение
        Результат.Ошибка = ПодробноеПредставлениеОшибки(ИнформацияОбОшибке());
    КонецПопытки;

    Возврат Результат;

КонецФункции
```

### Таймауты

**Обязательно** устанавливать таймаут соединения и чтения. Значения — из конфигурации (константы или настройки), не хардкод.

## Сериализация

### JSON

```bsl
// Запись — через платформу, НЕ ручная сборка строк
ЗаписьJSON = Новый ЗаписьJSON;
ЗаписьJSON.УстановитьСтроку(Новый ПараметрыЗаписиJSON(ПереносСтрокJSON.Unix));
ЗаписатьJSON(ЗаписьJSON, ДанныеДляОтправки);
СтрокаJSON = ЗаписьJSON.Закрыть();

// Чтение — в Соответствие (не в Структуру)
ЧтениеJSON = Новый ЧтениеJSON;
ЧтениеJSON.УстановитьСтроку(ТелоОтвета);
Данные = ПрочитатьJSON(ЧтениеJSON, Истина);
ЧтениеJSON.Закрыть();
```

### XML

Использовать `ЧтениеXML`/`ЗаписьXML` с XSD-валидацией (при наличии схемы).

### Форматы данных

| Тип | Формат | Примечание |
|-----|--------|------------|
| Даты | ISO 8601 (`YYYY-MM-DDTHH:MM:SSZ`) | Согласовать часовой пояс |
| Числа | Точка как разделитель | Проверять при чтении |
| Булевы | `true`/`false` | Не "да"/"нет" |

## Безопасность

### Хранение учётных данных

| Что | Где хранить |
|-----|------------|
| Пароли, токены, API-ключи | БСП "Безопасное хранение паролей" или защищённые константы |
| Адреса серверов | Константы или регистры сведений |
| Сертификаты | Хранилище сертификатов ОС |

**ЗАПРЕЩЕНО:** хардкод учётных данных в коде.

### Токены

- Проверять срок действия перед каждым запросом
- Централизованное обновление (refresh)
- Хранить в безопасном хранилище

## Идемпотентность и ретраи

### Ключ идемпотентности

```bsl
// Хранить ключи операций в регистре сведений
// Перед отправкой — проверить статус
СтатусОперации = ПолучитьСтатусОперации(КлючОперации);
Если СтатусОперации = "Выполнено" Тогда
    Возврат; // Не отправлять повторно
КонецЕсли;
```

### Политика повторов

- Ограниченное число попыток (3-5)
- Экспоненциальная задержка между попытками
- Повторять только для сетевых ошибок и 5xx, не для 4xx

---

## Длительные сетевые операции

Сетевые вызовы — через БСП `ДлительныеОперации.ВыполнитьФункцию()`.

**Запрещено:** прямой вызов `ФоновыеЗадания` для сетевых операций.

На клиенте — асинхронный паттерн с обработкой завершения.
