# Платформенные решения и шаблоны

> Применяется к: `**/*.bsl`, `**/*.os`

## Назначение

Каталог канонических шаблонов для типовых платформенных операций. Использовать как справочник при написании кода.

---

## 1. Проверка пустых значений

### Проблема

Прямое сравнение с пустыми ссылками или `Неопределено` может некорректно работать для составных типов.

### Шаблон

```bsl
// ПРАВИЛЬНО — универсальная проверка
Если ЗначениеЗаполнено(Контрагент) Тогда
    // Работа с заполненным значением
КонецЕсли;

// НЕПРАВИЛЬНО — прямое сравнение
Если Контрагент <> Неопределено Тогда   // Может не сработать для пустых ссылок
Если Контрагент <> "" Тогда              // Некорректно для ссылочных типов
```

## 2. Длительные операции

### Проблема

Операции дольше 10 секунд блокируют UI.

### Шаблон (с использованием БСП)

```bsl
// Серверная функция — запуск в фоне через БСП
&НаСервере
Функция ЗапуститьОбработкуДанных()

    ПараметрыВыполнения = ДлительныеОперации.ПараметрыВыполненияФункции(УникальныйИдентификатор);
    ПараметрыВыполнения.ОжидатьЗавершение = 0; // Не ждать

    Возврат ДлительныеОперации.ВыполнитьФункцию(
        ПараметрыВыполнения,
        "МойМодуль.ОбработатьДанныеВФоне",
        ПараметрыОбработки);

КонецФункции

// Клиентская обработка завершения
&НаКлиенте
Процедура ПослеВыполненияОперации(Результат, ДополнительныеПараметры) Экспорт

    Если Результат = Неопределено Тогда
        Возврат;
    КонецЕсли;

    Если Результат.Статус = "Выполнено" Тогда
        ОбработатьРезультат(Результат.АдресРезультата);
    ИначеЕсли Результат.Статус = "Ошибка" Тогда
        ПоказатьПредупреждение(, Результат.КраткоеПредставлениеОшибки);
    КонецЕсли;

КонецПроцедуры
```

**Запрещено:** Прямой вызов `ФоновыеЗадания.Выполнить()` — использовать БСП обёртку.

## 3. Транзакции

### Каноническая транзакция

```bsl
НачатьТранзакцию();
Попытка

    // Бизнес-логика внутри транзакции
    ДокументОбъект = Документ.ПолучитьОбъект();
    ДокументОбъект.Статус = НовыйСтатус;
    ДокументОбъект.Записать();

    ЗафиксироватьТранзакцию();

Исключение
    ОтменитьТранзакцию();
    ВызватьИсключение; // или логирование + обработка
КонецПопытки;
```

### Важно: вложенные транзакции

Обработчики событий (`ПередЗаписью`, `ПриЗаписи`) уже выполняются внутри транзакции записи объекта. **Не открывать вложенные транзакции** в обработчиках — управлять транзакцией в вызывающем коде.

## 4. Управляемые блокировки

### Шаблон при проведении документа

```bsl
Процедура Проведение(Документ, Отказ, РежимПроведения)

    // 1. Установить управляемую блокировку ПЕРЕД чтением остатков
    Блокировка = Новый БлокировкаДанных;
    ЭлементБлокировки = Блокировка.Добавить("РегистрНакопления.ТоварыНаСкладах");
    ЭлементБлокировки.Режим = РежимБлокировкиДанных.Исключительный;
    ЭлементБлокировки.УстановитьЗначение("Склад", Документ.Склад);
    Блокировка.Заблокировать();

    // 2. Теперь читать остатки — данные защищены
    ТекущиеОстатки = ПолучитьОстатки(Документ.Склад);

    // 3. Проверка и движения
    // ...

КонецПроцедуры
```

### Каноничный порядок блокировок

Для предотвращения взаимоблокировок — блокировать регистры в **одинаковом порядке** во всех документах.

## 7. Поиск в коллекциях

| Размер коллекции | Метод | Сложность |
|-----------------|-------|-----------|
| < 100 элементов | `Найти()` / линейный поиск | O(n) — допустимо |
| ≥ 100 элементов или в цикле | `Соответствие` для индексации | O(1) |

```bsl
// Для больших коллекций — индекс через Соответствие
Индекс = Новый Соответствие;
Для Каждого Элемент Из КоллекцияДанных Цикл
    Индекс.Вставить(Элемент.Ключ, Элемент);
КонецЦикла;

// Поиск — O(1)
НайденныйЭлемент = Индекс.Получить(ИскомыйКлюч);
```

## 8. Безопасный доступ к свойствам структуры

```bsl
// ПРАВИЛЬНО — проверка через Свойство()
Если Параметры.Свойство("Дата") Тогда
    Дата = Параметры.Дата;
КонецЕсли;

// Или с переменной
ЗначениеДаты = Неопределено;
Если Параметры.Свойство("Дата", ЗначениеДаты) Тогда
    // Использовать ЗначениеДаты
КонецЕсли;

// НЕПРАВИЛЬНО — обращение без проверки
Дата = Параметры.Дата; // Исключение если свойства нет
```

## 13. Ранний выход (Early Return)

```bsl
// ПРАВИЛЬНО — плоская структура
Функция ОбработатьЗаказ(Заказ)

    Если НЕ ЗначениеЗаполнено(Заказ) Тогда
        Возврат Неопределено;
    КонецЕсли;

    Если Заказ.Статус = "Отменен" Тогда
        Возврат Неопределено;
    КонецЕсли;

    // Основная логика
    Возврат Результат;

КонецФункции
```
