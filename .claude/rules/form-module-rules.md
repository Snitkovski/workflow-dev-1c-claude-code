# Правила модулей форм

> Применяется к: модулям управляемых форм

## Принципы клиент-серверного взаимодействия

### Главное правило

> **Минимизировать количество серверных вызовов.**
> Объединять несколько операций в один вызов.

---

## Директивы компиляции

### Обзор директив

| Директива | Выполнение | Контекст формы | Когда использовать |
|-----------|------------|----------------|-------------------|
| `&НаКлиенте` | Клиент | Да | Интерактивные действия, UI |
| `&НаСервере` | Сервер | Да | Изменение реквизитов формы |
| `&НаСервереБезКонтекста` | Сервер | Нет | Чтение данных, вычисления |
| `&НаКлиентеНаСервереБезКонтекста` | Оба | Нет | Утилиты, валидация |

### Правило выбора

```
&НаСервереБезКонтекста > &НаСервере
```

**Используйте `&НаСервере` ТОЛЬКО когда:**
- Нужно изменить реквизиты формы
- Нужно изменить видимость/доступность элементов
- Нужен доступ к `ЭтотОбъект`

**Используйте `&НаСервереБезКонтекста` для:**
- Получения данных из БД
- Вычислений на сервере
- Валидации
- Любых операций, не требующих контекст формы

## Оптимизация серверных вызовов

### Объединение вызовов

```bsl
// ❌ НЕПРАВИЛЬНО — 3 серверных вызова
&НаКлиенте
Процедура ЗаполнитьДанные()
    ДанныеКонтрагента = ПолучитьДанныеКонтрагентаНаСервере(Контрагент);
    ДанныеДоговора = ПолучитьДанныеДоговораНаСервере(Договор);
    Цены = ПолучитьЦеныНаСервере(СписокТоваров);
КонецПроцедуры

// ✅ ПРАВИЛЬНО — 1 серверный вызов
&НаКлиенте
Процедура ЗаполнитьДанные()
    ВсеДанные = ПолучитьВсеДанныеНаСервере(Контрагент, Договор, СписокТоваров);

    ДанныеКонтрагента = ВсеДанные.Контрагент;
    ДанныеДоговора = ВсеДанные.Договор;
    Цены = ВсеДанные.Цены;
КонецПроцедуры

&НаСервереБезКонтекста
Функция ПолучитьВсеДанныеНаСервере(Контрагент, Договор, СписокТоваров)
    Результат = Новый Структура;
    Результат.Вставить("Контрагент", ПолучитьДанныеКонтрагента(Контрагент));
    Результат.Вставить("Договор", ПолучитьДанныеДоговора(Договор));
    Результат.Вставить("Цены", ПолучитьЦены(СписокТоваров));
    Возврат Результат;
КонецФункции
```

### Серверные вызовы в циклах

```bsl
// ❌ КРИТИЧЕСКИ НЕПРАВИЛЬНО — N серверных вызовов
&НаКлиенте
Процедура ОбработатьСтроки()
    Для Каждого Строка Из Объект.Товары Цикл
        Цена = ПолучитьЦенуНаСервере(Строка.Номенклатура); // N вызовов!
        Строка.Цена = Цена;
    КонецЦикла;
КонецПроцедуры

// ✅ ПРАВИЛЬНО — 1 серверный вызов
&НаКлиенте
Процедура ОбработатьСтроки()
    СписокНоменклатуры = Новый Массив;
    Для Каждого Строка Из Объект.Товары Цикл
        СписокНоменклатуры.Добавить(Строка.Номенклатура);
    КонецЦикла;

    Цены = ПолучитьЦеныПакетомНаСервере(СписокНоменклатуры); // 1 вызов

    Для Каждого Строка Из Объект.Товары Цикл
        Строка.Цена = Цены.Получить(Строка.Номенклатура);
    КонецЦикла;
КонецПроцедуры
```

## Структура модуля формы

```bsl

// Клиентские переменные формы
Перем КешДанных;

#КонецОбласти

#Область ОбработчикиСобытийФормы

    // Инициализация формы
    Если Параметры.Свойство("КлючНазначенияИспользования") Тогда
        // ...
    КонецЕсли;

&НаКлиенте
Процедура ПриОткрытии(Отказ)
    // Действия при открытии

&НаКлиенте
Процедура ПередЗакрытием(Отказ, ЗавершениеРаботы, ТекстПредупреждения, СтандартнаяОбработка)

#КонецОбласти

#Область ОбработчикиСобытийЭлементовШапкиФормы

Процедура КонтрагентПриИзменении(Элемент)
    КонтрагентПриИзмененииНаСервере();
КонецПроцедуры

&НаСервере
Процедура КонтрагентПриИзмененииНаСервере()
    // Заполнение зависимых реквизитов

#КонецОбласти

#Область ОбработчикиСобытийЭлементовТаблицыФормыТовары

&НаКлиенте
Процедура ТоварыНоменклатураПриИзменении(Элемент)
    ОбработатьИзменениеНоменклатуры();
КонецПроцедуры

&НаКлиенте
Процедура ТоварыКоличествоПриИзменении(Элемент)
    РассчитатьСумму();
КонецПроцедуры

#КонецОбласти

#Область ОбработчикиКомандФормы

&НаКлиенте
Процедура Заполнить(Команда)
    ЗаполнитьНаСервере();
КонецПроцедуры

&НаСервере
Процедура ЗаполнитьНаСервере()
    // Логика заполнения
КонецПроцедуры

#КонецОбласти

#Область СлужебныеПроцедурыИФункции

// --- Клиентские ---

&НаКлиенте
Процедура ОбработатьИзменениеНоменклатуры()
    // Клиентская логика

&НаКлиенте
Процедура РассчитатьСумму()
    ТекущаяСтрока = Элементы.Товары.ТекущиеДанные;
    Если ТекущаяСтрока <> Неопределено Тогда
        ТекущаяСтрока.Сумма = ТекущаяСтрока.Количество * ТекущаяСтрока.Цена;
    КонецЕсли;
КонецПроцедуры

// --- Серверные с контекстом ---

&НаСервере
Процедура ОбновитьВидимостьЭлементов()
    // Управление видимостью
КонецПроцедуры

// --- Серверные без контекста ---

&НаСервереБезКонтекста
Функция ПолучитьДанныеНоменклатуры(Номенклатура)
    Возврат ОбщегоНазначения.ЗначенияРеквизитовОбъекта(
        Номенклатура,
        "Наименование, ЕдиницаИзмерения, Артикул");
КонецФункции

// --- Клиент-серверные ---

&НаКлиентеНаСервереБезКонтекста
Функция ПроверитьЗаполненность(Значение)
    Возврат ЗначениеЗаполнено(Значение);
КонецФункции

#КонецОбласти
```

## Асинхронное программирование

### Современный подход (8.3.18+)

```bsl
// ✅ ПРАВИЛЬНО — асинхронные методы
&НаКлиенте
Асинх Процедура ВыбратьФайл(Команда)

    Результат = Ждать ПоместитьФайлНаСерверАсинх(, , , Истина);

    Если Результат.Помещен Тогда
        ОбработатьФайлНаСервере(Результат.Адрес);
    КонецЕсли;

КонецПроцедуры

&НаКлиенте
Асинх Процедура ПоказатьВопросИПродолжить()

    Ответ = Ждать ВопросАсинх("Продолжить?", РежимДиалогаВопрос.ДаНет);

    Если Ответ = КодВозвратаДиалога.Да Тогда
        ВыполнитьОперацию();
    КонецЕсли;

КонецПроцедуры
```

### Устаревший подход (до 8.3.18)

```bsl
// ⚠️ УСТАРЕВШИЙ — ОписаниеОповещения
&НаКлиенте
Процедура ВыбратьФайл(Команда)

    ОписаниеОповещения = Новый ОписаниеОповещения(
        "ВыбратьФайлЗавершение",
        ЭтотОбъект);

    НачатьПомещениеФайла(ОписаниеОповещения, , , Истина);

КонецПроцедуры

&НаКлиенте
Процедура ВыбратьФайлЗавершение(Результат, Адрес, ВыбранноеИмяФайла, ДополнительныеПараметры) Экспорт

    Если Результат Тогда
        ОбработатьФайлНаСервере(Адрес);
    КонецЕсли;

КонецПроцедуры
```

**Правило:** Предпочитать `Асинх`/`Ждать` для версий 8.3.18+.

## Преобразование данных формы

### Работа с объектом на сервере

```bsl
&НаСервере
Процедура ОбработатьОбъект()

    // Преобразование данных формы в объект
    ДокументОбъект = ДанныеФормыВЗначение(Объект, Тип("ДокументОбъект.ЗаказПокупателя"));

    // Работа с объектом
    ДокументОбъект.Заполнить();

    // Преобразование обратно в данные формы
    ЗначениеВДанныеФормы(ДокументОбъект, Объект);

КонецПроцедуры
```

### Работа с табличными частями

```bsl
&НаСервере
Процедура ОбработатьТабличнуюЧасть()

    // Преобразование табличной части в таблицу значений
    ТаблицаТоваров = Объект.Товары.Выгрузить();

    // Обработка
    Для Каждого Строка Из ТаблицаТоваров Цикл
        // ...
    КонецЦикла;

    // Загрузка обратно
    Объект.Товары.Загрузить(ТаблицаТоваров);

КонецПроцедуры
```

## Типичные ошибки

### 1. Серверный вызов в обработчике изменения

```bsl
// ❌ НЕПРАВИЛЬНО — серверный вызов для каждой строки
&НаКлиенте
Процедура ТоварыНоменклатураПриИзменении(Элемент)
    Данные = ПолучитьДанныеНоменклатурыНаСервере(Номенклатура); // Медленно!
КонецПроцедуры

// ✅ ПРАВИЛЬНО — кешировать или получать пакетом
&НаКлиенте
Процедура ТоварыНоменклатураПриИзменении(Элемент)
    // Получить из кеша или запросить один раз при открытии
    Данные = КешНоменклатуры.Получить(Номенклатура);
КонецПроцедуры
```

### 2. Обработка всех строк на сервере при изменении одной

```bsl
// ❌ НЕПРАВИЛЬНО — пересчёт всех строк
&НаСервере
Процедура ТоварыКоличествоПриИзмененииНаСервере()
    Для Каждого Строка Из Объект.Товары Цикл
        Строка.Сумма = Строка.Количество * Строка.Цена;
    КонецЦикла;
КонецПроцедуры

// ✅ ПРАВИЛЬНО — пересчёт на клиенте
&НаКлиенте
Процедура ТоварыКоличествоПриИзменении(Элемент)
    ТекущаяСтрока = Элементы.Товары.ТекущиеДанные;
    ТекущаяСтрока.Сумма = ТекущаяСтрока.Количество * ТекущаяСтрока.Цена;
КонецПроцедуры
```

### 3. Использование &НаСервере для чтения

```bsl
// ❌ НЕПРАВИЛЬНО — передача контекста для чтения
&НаСервере

    // Чтение данных не требует контекста формы
КонецФункции

// ✅ ПРАВИЛЬНО
&НаСервереБезКонтекста
Функция ПолучитьОстаток(Номенклатура, Склад)
    // Минимальный трафик
КонецФункции
```
